🔄 GitHub Actions 手动刷新

当前仓库：CharmingLyy/investment-tracker
部署地址：https://charminglyy.github.io/investment-tracker/

💡 使用说明：
1. 在对应标签页添加/删除标的
2. 点击 💾 保存到本地 — 浏览器会记住你的配置
3. 点击 📥 下载 watchlist.json，放入 data/ 目录
4. 提交到 GitHub 仓库，等待 Actions 自动更新后刷新页面
5. 或者点击 🚀 触发刷新工作流 直接在线触发更新（需设置 Token）
